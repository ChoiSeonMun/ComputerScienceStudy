# Chapter 1 - 컴퓨터 내부의 언어체계
---

### 비트
- 비트는 Binary와 Digit의 합성어이다.
- 비트에는 2가지 정보를 담을 수 있다.
    - 정보는 내가 원하는 대로 하면 된다.

### 정수 표현
- 비트로 정수를 표현하려면 2진수를 사용한다.
- 2진수는 보통 8진수나 16진수 표현법을 사용한다.
- 음수는 2의 보수를 사용한다.

### 실수 표현
- 실수는 IEEE 754 표준을 사용한다.
- 기본 정밀도와 더블 정밀도가 있다.
- 지수부와 가수부로 나눠서 표현한다.

### 문자 표현
- 문자는 ASCII를 주로 사용한다.
- 요새는 다국어를 지원하기 위해 유니코드를 사용한다.
    - 현재 가장 많이 사용되고 있는 유니코드는 UTF-8이다.
